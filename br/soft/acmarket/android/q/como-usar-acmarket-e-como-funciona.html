Como usar ACMarket e como funciona

WhatsApp Compartilhar Tweet Enviar

Veja o passo a passo para usar esta popular loja de aplicativos para você conhecer completamente todas suas funções. Além disso, explicamos a seguir como se registrar nele

ACMarket é um enorme mercado de aplicativos. Muitos deles foram modificados para obter certas funções ou benefícios pagos de forma totalmente grátis. Além disso, encontramos outro tipo de conteúdo, como livros. Por outro lado, o próprio aplicativo integra algumas funções úteis. Por exemplo, temos um scanner de vírus ou um otimizador de armazenamento.

Para começar a usar o ACMarket você deverá baixar o apk. Pode ser feito diretamente desde esta página, onde você encontrará a versão mais atual. Depois de instalar e abrir, ACMarket nos dará as boas-vindas solicitando algumas permissões. Para continuar clique OK.

Tela de boas-vindas

Agora, o aplicativo solicitará permissões para fazer chamadas. Isso não será necessário, recomendamos não aceitar. Em qualquer caso, selecione Allow para aceitar e Deny para rejeitar.

Permissão de gerenciamento de chamadas

A próxima permissão que precisamos aceitar é o acesso ao armazenamento. Ele é necessário. Caso contrário, ACMarket não poderia baixar aplicativos em nosso dispositivo. Escolha Allow para aceitar.

Permite acesso ao armazenamento

Agora você já está na tela inicial do ACMarket.

Tela inicial

Embora não seja necessário, para acessar algumas funções extras é necessário ter um usuário. Explicamos como criar um.

Como criar uma conta no ACMarket

Primeiro, abra o menu lateral clicando no ícone de usuário.

Abrir menu de usuário

Agora clique em Login.

Menu inicial de sessão

Insira seu e-mail e pressione Continue.

Iniciar login no ACMarket

Escolha um pseudônimo. Use o botão Continue para continuar com o processo.

Escolher um nick

Selecione seu país. Clique em Continue para prosseguir.

Selecionar país

Indique sua idade e clique Continue.

Selecionar idade

Escolha seu gênero. Use o botão Continue para proceder.

Selecionar gênero

ACMarket permite agora que você criar um avatar personalizado. Use as diferentes opções que você encontrará no canto inferior da tela para personalizá-lo à vontade. Depois de finalizar, clique em Finish.

Criar avatar

Crie uma senha. Ela deve ter no mínimo 6 caracteres e um máximo de 32. Clique uma última vez Continue.

Criar senha

Feito isso, agora você já tem sua conta de usuário no ACMarket que desbloqueia algumas funções exclusivas.

Como baixar e instalar aplicativos com ACMarket

A principal missão do ACMarket é oferecer downloads de software ao usuário. É muito simples começar a baixar conteúdo. Para começar, localize aquele aplicativo que você deseja instalar. Tomaremos como exemplo o WhatsApp, um aplicativo muito popular e que não foi modificado por ninguém. Na tela principal, localize a barra de pesquisa.

Buscador do ACMarket

Digite WhatsApp e clique intro.

Buscar o WhatsApp

Entre uma grande lista de utilitários e aplicativos, podemos encontrar a versão oficial do WhatsApp. Para começar o download, clique no ícone da seta à direita.

Começar o download

Clique em Free Download.

Download gratuito do WhatsApp no ACMarket

Depois de visualizar um anúncio, o download começará. Você pode checar seu status na barra de notificações.

Progresso do download de um aplicativo

Assim que tivermos o apk, a instalação começa automaticamente. Clique em Install para continuar.

Instalar o WhastApp

Se visualizamos a mensagem App Installed é que tudo correu como planejado. Selecione a opção Open para abrir o aplicativo.

Abrir aplicativo recém instalado

Como funcionam o resto das funções do ACMarket

ACMarket tem algumas outras funções muito boas. Seguidamente, fazemos uma breve revisão pelo aplicativo para você aprender como ele funciona.

Na tela inicial teremos acesso às categorias com maior destaque.

Categorias mais destacadas

Também podemos acessar o gerenciador de downloads clicando no canto superior direito.

Acessar gerenciador de downloads

Nessa seção é possível checar quais aplicativos instalamos e se tem atualizações disponíveis. Também temos um resumo do status do armazenamento.

Gerenciador de downloads

Novamente na tela principal, focamos nossa atenção na barra inferior. Dispomos de um acesso rápido no início denominado Cake, a um fórum onde os usuários participam, Topic, e um minigame chamado Bingo.

Menu inferior

No entanto, é no menu de usuário onde encontramos a maior parte das funções do ACMarket. Para abrir, devemos clicar sobre nosso avatar.

Menu de usuário

No menu encontramos uma série de opções que atualmente não estão disponíveis. Trata-se da opção Become a VIP ou da carteira virtual que lança mensagens como This function is not avalible now.

Funções não disponíveis

As seções Apps and Games e App update nos levam ao mesmo lugar: o gerenciador de downloads que acessamos na tela inicial.

Acessos ao gerenciador de downloads no menu de usuário

A opção Gift code permite trocar códigos, App request mostra um formulário para solicitar aplicativos que faltem e Settings permite modificar alguns comportamentos adicionais.

Outras seções de ACMarket

E finalmente, Phone Assistant merece nossa atenção. Essa seção se apresenta como um pequeno canivete suíço no qual podemos limpar o armazenamento do nosso telefone, checar se os apps instalados têm vírus ou consultar aqueles que marcamos como favoritos.

Função assistente do telefone

Para iniciar uma varredura dos apps que instalamos com ACMarket, clique em Virus Scan.

Scanner de vírus

Em seguida, selecione o aplicativo que deseja escanear.

Escolher app para escanear

Na tela obterá um completo relatório com as informações de diferentes sites voltados à detecção de vírus.

Relatório completo do WhatsApp

Quanto ao otimizador de armazenamento, clique em System Cleaner para rodar.

Limpador de armazenamento

Clique em Clean all e ACMarket limpará aqueles dados desnecessários.

Limpar arquivos desnecessários do ACMarket
