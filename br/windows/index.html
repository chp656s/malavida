Microsoft Windows

Windows é o sistema operacional (SO) da Microsoft criado em 1985 e famoso por sua interface gráfica de usuário baseada em janelas (GUI – Graphical user interface). Este sistema operacional não roda somente nos computadores pessoais, mas também em dispositivos móveis e servidores em distintas versões, sendo o SO mais instalado no mundo.

Com o Windows 10, a empresa de Redmond fez um esforço para ter somente um sistema operacional para todos os dispositivos, para que um usuário com um smartphone e outro com um computador de mesa tenham a mesma interface, embora adaptada ao seu dispositivo.
