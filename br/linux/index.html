Linux OS

Linux é sinônimo de software livre, um ambiente de trabalho em constante evolução cujo código-fonte pode ser usado e distribuído livremente. Linus Torvalds foi o principal propulsor desta plataforma que hoje conta com algumas distribuições bem populares, como Ubuntu, Fedora, Debian ou openSUSE, utilizados tanto para a gestão empresarial quanto para o lazer e as tarefas domésticas.

Algumas distribuições Linux contam com seu próprio programa de instalação e atualização de software, mas alguns desenvolvedores preferem distribuir o seu software de forma independente.

Com a chegada de projetos como o Steam para sistemas Linux, estes se tornaram uma boa alternativa para os gamers que buscam plataformas seguras e livres de malware para jogos online.
