Daemon Tools Виртуальный диск ru

Daemon Tools lite - это бесплатная программа для создания виртуального cd/dvd/blu-rey привода на вашем компьютере, а также для создания виртуального образа диска, с расширением файла *.iso и записи образа диска на реальный диск (балванку).

Многие спрашивает, как установить игру которую я скачал(а) с интернета, игра одним файлов расширение *.iso. Установить такую игру можно двумя способами, записать образ на диск, или создать виртуальный диск с помощью программы Daemon Tools и установить игру. Для создания виртуального диска, нужно скачать бесплатную программу Daemon Tools lite и установить её. После установки и перезагрузки windows, у Вас в компьютере появится дополнительный дисковод. Теперь нужно смонтировать образ диска(игру которую вы скачали) с программой Daemon Tools. Для этого, нажимаем в трее(в правом нижнем углу возле часов) на ярлык Daemon Tools два раза левой кнопкой мыши, у вас откроется программа, в ней нажмите на кнопку "Добавить образ"(Диск с +) выберите Вашу игру и нажмите открыть. После этих действий, если игра с автоспуском, начнется установки игры, если игра без автозапуска, нажмите в программе Daemon Tools на диск с названием игры два раза правой кнопкой мышки.

Программа Daemon Tools lite - это бесплатная программа, без ключей и регистрации, при установки программы, нужно во втором окне нажать на ссылку "Продолжить с бесплатной лицензией".
