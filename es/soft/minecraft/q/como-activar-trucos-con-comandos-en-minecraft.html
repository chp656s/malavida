Cómo activar trucos con comandos en Minecraft para PC

WhatsApp Compartir Tweet Enviar

Los trucos de Minecraft proporcionan diferentes ventajas pero claro, de nada sirven si no sabes cómo activarlos. Te mostramos la manera de hacerlo y los comandos que debes teclear en cada caso

Minecraft es un juego que permite activar toda clase de trucos gracias a comandos. Estos comandos se activan desde la barra de chat del juego, te explicaremos cómo hacerlo. Eso sí, activar trucos con comandos en Minecraft es una opción que tienes que haber permitido en el momento en el que creas un mundo nuevo. Solo entonces aparece esa opción. Si no la marcas, después te será imposible hacerlo.

Crea un nuevo mundo con los trucos activados

Es importante saber que si se marca esta opción no se podrá desbloquear ninguno de los logros, ya que se considera que se adultera la partida.

Con tu mundo nuevo creado con los trucos habilitados, accede a la partida. Una vez en ella pulsa la tecla T en tu teclado, verás cómo se abre un pequeño editor de texto en la parte inferior de la pantalla.

Pulsa la tecla T para abrir la ventana de comandos

Ahora solo tienes que introducir el comando del truco que quieras emplear. Por ejemplo, uno de los comandos habituales es el que permite alternar entre el modo supervivencia de juego y el modo creativo. Como imaginarás, en el primero tu personaje puede morir, en el segundo no. Teclea /gamemode creative y pulsa la tecla Enter para pasar al modo creativo.

Escribe el comando que quieras

En cuanto el truco se active recibirás en la misma pantalla un aviso.

Aviso de que el comando se ha introducido correctamente

Como has podido ver, al teclear la barra inclinada el mismo juego te ofrece sugerencias de trucos que puedes aplicar.

Por si puede resultarte de ayuda, te dejamos aquí todos los trucos de Minecraft existentes a través de la consola de comandos. Todos los trucos empiezan con una barra inclinada a la que siguen diferentes instrucciones., en algunos casos más específicas que en otros.

Comandos para cambiar el modo de juego

  • /gamemode survival: Activa el modo de juego supervivencia
  • /gamemode creative: Activa el modo de juego creativo
  • /gamemode adventure: Activa el modo de juego aventura
  • /gamemode spectator: Activa el modo de juego espectador

Comandos para cambiar la dificultad

  • /difficulty peaceful: elimina por completo los peligros y por tanto la dificultad
  • /difficulty easy: modo de dificultad fácil
  • /difficulty normal: modo de dificultad normal
  • /difficulty hard: modo de dificultad difícil

Comandos para cambiar la hora del día

  • /gamerule doDaylightCycle false: para el transcurso del tiempo
  • /time set day: establece la hora de juego a día
  • /time set night: establece la hora de juego a la noche
  • /time set 18000: establece la hora de juego a medianoche
  • /time set 6000: establece la hora de juego a mediodía
  • /time set 12000: establece la hora de juego al anochecer
  • /time set 0: establece la hora de juego al amanecer
  • /time query gametime: resetea la hora de juego

Comandos para cambiar el clima

  • /weather rain: provoca la lluvia
  • /weather thunder: provoca una tormenta con truenos
  • /weather clear: para despejar el cielo
  • /gamerule doWeatherCycle false: mantiene el clima actual

Otros comandos y trucos

  • /enchant [nombrepersonaje] [nombreencantamiento] [nivelencantamiento]: para el encantamiento de objetos
  • /summon [nombre del enemigo] [coordenadas]: para invocador personajes y NPCs
  • /teleport [nombre del personaje] [coordenadas]: permite el teletransporte del jugador a otras coordenadas del mapa
  • /ability: realiza modificaciones en la habilidad de algún jugador
  • /advancement: comprueba y modifica los progresos de los jugadores
  • /attribute: para gestionar los atributos de una entidad
  • /ban: permite banear a un jugador
  • /ban-ip: permite banear a un jugador usando su dirección IP
  • /banlist: hace visible la lista de jugadores baneados
  • /bossbar: para crear y modificar bossbars, que aparecen cuando nos enfrentamos a algún jefe
  • /camerashake: activa un efecto de cámara temblando
  • /classroommode: permite editar bloques restringidos
  • /clear: borra el inventario del jugador
  • /clone: permite copiar bloques en diferentes lugares
  • /code: para acceder a la conexión de código
  • /collect: habilita la recolección de todos los objetos dentro de un bloque para el MOB Agente en tres dimensiones
  • /createagent: crea un nuevo agente allá donde estés
  • /data: obtiene, fusiona, modifica y elimina datos de entidades de bloque y entidades NBT
  • /datapack: permite controlar los paquetes de datos cargados
  • /debug: empieza o concluye una sesión de depuración
  • /defaultgamemode: aplica el modo de juego predeterminado
  • /deop: revoca el estado de operador de cualquier jugador
  • /effect: quita o pone efectos de estado
  • /fill: usa un bloque específico para rellenar una región al completo
  • /forceload: fuerza la carga de chunks siempre o al contrario, nunca
  • /function: ejecuta una función
  • /inmutableworld: establece inmutabilidad de un mundo
  • /kill nombrejugador: permite el suicidio o matar a un jugador específico
  • /loot: tira los objetos del inventario al suelo
  • /msg: envía mensajes privados a otros jugadores
  • /op: da estatus de operador a un jugador
  • /pardon: remueve entradas de la lista negra
  • /particle: para crear partículas
  • /gamerule keepInventory true: permite conservar todo el inventario después de morir
  • /experience add: añade experiencia a tu jugador
  • /dropstore: pone todos los objetos del inventario en un cofre
  • /help NombreComando: para obtener información sobre cualquier comando
  • /freeze: paraliza todas las entidades del juego
  • /instantplant: hace crecer las plantas al momento
  • /waterdamage: activa o desactiva daño por agua
  • /falldamage: activa o desactiva el daño por caída
  • /firedamage: activa o desactiva el daño por fuego
  • /itemdamage: activa o desactiva el desgaste en los objetos
  • /duplicate: para copiar el objeto que tenemos en la mano en ese momento
  • /superheat: sirve para llevar todos los objetos posibles a su forma fundida
  • /atlantis: hace subir mucho el nivel de los mares y océanos
  • /ride: permite convertir cualquier criatura a la que estés mirando en una criatura montable
  • /instantmine: permite romper bloques con un golpe con independencia de la herramienta
  • /seed: ofrece el código correspondiente a la semilla del mundo en el que estamos
  • /raytracefog: para ajustes gráficos de niebla en el ray tracing
  • /save: para preparar una copia de seguridad o ver su estado
  • /save-all: guarda el servidor
  • /schedule: retrasa la ejecución de una función
  • /setmaxplayers: marca el número máximo de jugadores permitidos para entrar
  • /spreadplayers: para transportar entidades a ubicaciones aleatorias
  • /stopsound: parar el sonido
  • /worldborder: administra el borde del mundo
  • /worldbuilder: permite la edición de bloques restringidos
