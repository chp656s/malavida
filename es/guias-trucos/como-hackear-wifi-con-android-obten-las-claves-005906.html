Cómo hackear una red Wifi con Android

WhatsApp Compartir Twittear Enviar
Cómo hackear redes wifi, descifrar sus contraseñas y saltarse su seguridad: aprende a hacer todo esto para averiguar si tu red wifi es vulnerable a ataques, robo de wifi o conexiones indeseadas
Android Seguridad y Privacidad Android Conectividad
Por Rocío Gonzálezel 19 enero, 2016 (Actualizado el 28 enero, 2021)
en Guías y Trucos

Aprende a descifrar claves wifi desde tu dispositivo Android de una forma sencilla y segura. Para ello, vamos a hablar de algunas aplicaciones Android muy fáciles de utilizar con las que podremos hackear conexiones wifi con distintos niveles de seguridad como contraseñas WEP, WPA, WPA2 o WPA2-PSK. Prepara tu dispositivo Android y prepárate para probar con esta selección de aplicaciones gratuitas que funcionan como herramientas para poder hackear las claves wifi de las conexiones que te rodean. No son programas para robar wifi, estas aplicaciones legales sirven para un uso responsable, como comprobar la seguridad de nuestra propia red y cómo de protegidos estamos ante la intrusión de usuarios externos y ataques de hackers.

Descubrir claves wifi con Android, más fácil de lo que crees

El amor es como el Wi-Fi: está en el aire pero no todo el mundo tiene la clave

Conseguir las claves wifi es sencillo, y hacerlo a través de un dispositivo Android más, gracias a las aplicaciones específicas que encontramos en Google Play, totalmente legales y muy fáciles de utilizar. En pocos pasos podrás comprobar la seguridad de las claves de tu alrededor y si tú conexión necesita un poco más de atención. Además, si has perdido tu contraseña, podrás recuperarla sin tener que depender de nadie.

Hackear contraseñas WEP, WPA, WPA2 y WPA2-PSK

Con estos programas podemos hackear redes wifi con contraseñas de diferentes niveles de seguridad como WEP, WPA, WPA2 y WPA2-PSK. ¿Conoces las diferencias entre ellas?

El más vulnerable es el protocolo WEP (Wired Equivalent Privacy), con un cifrado de nivel 2 y claves de 64 o 128 bits (10 o 26 caracteres hexadecimales).

Más seguro es el protocolo WPA (Wi-Fi Protected Access) con una clave de 128 bits que permite letras, números y caracteres especiales y el uso de TKIP, que cambia la clave de cifrado con cada paquete que se transmite. A la hora de elegir nuestra contraseña para nuestra red de casa, es importante conocer con qué seremos menos vulnerables.

Hoy en día lo más habitual, y casi sinónimo de invulnerabilidad es utilizar una contraseña WPA2 (o variante), aunque como la opción WEP continúa disponible en algunos routers, algunos la utilicen sin saber que podrían tener una mayor seguridad sin esfuerzo.

Aplicaciones Android para descifrar claves wifi

Prueba estas apps de Google Play para hackear claves wifi y medir la seguridad de tu propia red. ¿Estás protegido?

Router Keygen, descifra claves vulnerables

Router Keygen es una potente herramienta para hackear claves wifi vulnerables que nos permitirá desvelar si nuestra conexión puede ser objeto de ataques o conexiones de terceros. Para probarla, lo primero que tienes que hacer es descargarla e instalarla, después, ábrela desde tu dispositivo Android.

Router Keygen: escanea redes cercanas y accede a sus contraseñas wifi

Escanear redes cercanas. La primera tarea de la aplicaciones es hacer un barrido de las redes wifi cercanas y dividirlas entre soportadas y no soportadas. Las conexiones bajo la sección roja no son vulnerables por lo que Router Keygen no podrá facilitar sus contraseñas, sin embargo, el resto de secciones incluyen redes cuyas contraseñas se nos mostrarán si pulsamos en ella. Si apareces aquí, tu conexión necesita una contraseña mejor, tu ordenador un cortafuegos o tu router que delimites qué dispositivos pueden acceder a la red.

Conectar a una wifi vulnerable. Una vez elegida una wifi “soportada” podemos conectarnos directamente o copiar la contraseña para hacer el proceso de manera manual, dirigiéndonos al apartado "Ajustes> Wi-Fi" del menú de Android.
